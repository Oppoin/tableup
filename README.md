Data table that conforms to [Google Material Design](https://material.io/design/).

## Installation

```
$ npm install --save tableup
```

## Importing

```js
import TableUp from 'tableup';
import 'tableup/css/tableup.css';
```

## How to run demo

run `npm install` and then `npm run start`.

## How to upgrade npm

`npm install -g npm`
